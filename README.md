 Sistema de Combate RPG em C

Projeto simples desenvolvido para praticar lógica condicional em C.

 Funcionalidades

Sistema de vida para jogador e inimigo

Atributos de ataque e defesa

Cálculo de dano

Verificação de vitória, derrota ou empate

 Observação

O combate acontece em um único turno, pois ainda não foram implementadas estruturas de repetição.
O objetivo foi consolidar a base de lógica antes de avançar para loops e sistema de turnos.
